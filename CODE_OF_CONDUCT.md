# Code of Conduct

This project is a small open-source community space for Claude Code plugin
development. We want it to be useful, welcoming, and low-drama for contributors
at every experience level.

## Our Standards

Please help keep the project healthy by:

- Being respectful of different backgrounds, time zones, skill levels, and
  communication styles.
- Giving feedback that is specific, actionable, and focused on the work.
- Assuming good intent while still making room to correct mistakes quickly.
- Welcoming documentation fixes, tests, examples, and small improvements as
  valuable contributions.
- Keeping discussions on topic and moving extended design debates into issues or
  discussions when they need more room.

Unacceptable behavior includes harassment, discriminatory language, personal
attacks, deliberate disruption, publishing private information, or any other
conduct that makes participation unsafe or hostile.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer at
premprakashsharma.dev@gmail.com. Include links, screenshots, or context when
available. Reports will be handled privately and with care.

## Enforcement

Maintainers may remove comments, close threads, reject contributions, or block
participants when needed to protect the project community. Responses will be
proportional to the impact and pattern of behavior.

## Scope

This code of conduct applies to project spaces, including GitHub issues, pull
requests, discussions, reviews, and any other community channel created for this
repository.
