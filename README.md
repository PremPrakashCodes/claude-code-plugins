# Prem Prakash's Plugin Marketplace

A [Claude Code](https://claude.com/claude-code) plugin marketplace.

## Add this marketplace

```
/plugin marketplace add PremPrakashCodes/<this-repo>
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [status-line](plugins/status-line) | Configurable status-line HUD - model, project, git, context bar, and usage windows. Pure Python (stdlib only), selectable bar styles, themes, and custom labels. |
